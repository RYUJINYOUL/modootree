당신은 Gemini API 비용 분석 전문가입니다. 다음 서비스의 예상 비용을 계산해주세요.

[Gemini 2.5 Flash 사용 서비스]
1. AI 채팅 (ai-comfort)
   - 사용자당 하루 200회 제한
   - 예상 활성 사용자: 150명 (전체 회원 1000명 중 일일 활성 사용자)
   - 채팅으로 일기/메모/건강 등 모든 기능이 시작됨
   - 평균 입력: 프롬프트(약 500토큰) + 사용자 메시지(약 100토큰)
   - 평균 출력: 약 300토큰

2. 사연 요약 (ai-story-summary)
   - 일일 사용량: 약 20회
   - 평균 입력: 대화 내용(약 1000토큰)
   - 평균 출력: 약 500토큰

3. 건강 분석 초기 (health-analyze)
   - 일일 사용량: 약 100회
   - 평균 입력: 식사/운동 기록(약 300토큰)
   - 평균 출력: 약 200토큰

4. 건강 분석 최종 (health-final-analyze)
   - 일일 사용량: 약 100회
   - 평균 입력: 전체 기록(약 500토큰)
   - 평균 출력: 약 400토큰

[Gemini 2.5 Pro 사용 서비스]
1. 뉴스 크론 작업 (news/cron)
   - 하루 1회 실행
   - 4개 피드 × 4개 기사
   - 평균 입력: 기사당 약 1000토큰
   - 평균 출력: 기사당 약 800토큰

2. 사연/뉴스 투표 생성 (modoo-vote-ai-suggest)
   - 일일 사용량: 약 50회
   - 평균 입력: 약 800토큰
   - 평균 출력: 약 600토큰

3. 사진 스토리 생성 (photo-story/generate)
   - 일일 사용량: 약 50회
   - 이미지 + 프롬프트 입력
   - 4가지 모드 각각 생성
   - 평균 출력: 모드당 약 200토큰
